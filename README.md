# CS492_UserAuth

https://cs492-user-auth.herokuapp.com/login


## How to use it
```bash
$ # Get the code
$ git clone https://github.com/ChenYang-Lin/CS492_UserAuth.git
$ cd CS492_UserAuth
$
$ # Install node_modules 
$ npm install 
```
